关于返回顶部功能的实现

运用css+js实现

Step 1

关于这个功能的实现并不太难，只是有一些细节处理需要注意。如超出窗口高度自动淡出以及刷新时在主页面的淡出。

Step 2

首先建立一个button按钮，类名为back-to-top,css处理样式如下：

<button class="back-to-top">返回顶部</button> .back-to-top { background: rgba(255,255,255,0.4); font-size: 15px; font-family: "微软雅黑"; position: fixed; bottom:30px; right: 10px; border: 0.5px solid #888; }

Step 3

下面为使用js实现细节功能：

;$(function() { 'use strict'; backButton=$('.back-to-top'), backButton.on('click',function() { $('html,body').animate( /*上滑*/ { scrollTop:0 },800) /*速度*/ }) $(window).on('scroll',function() { if($(window).scrollTop()>$(window).height()) /*淡入淡出判断*/ backButton.fadeIn(); else backButton.fadeOut(); }) $(window).trigger('scroll');/*主页面不予显示*/ })
