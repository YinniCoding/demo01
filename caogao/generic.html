关于侧拉菜单功能的实现

运用css+js实现

Step 1

首先分析要实现的一些关键部分。页面暗化、侧拉菜单的出现、隐藏，以及之间衔接问题。

Step 2

接着是准备工作，百度cdn下载jQuery.js,并导入本地assets中。 在index.html中导入main.css,main.js以及jQuery.js
然后建两个div,分别制作页面暗化效果和侧拉菜单。便于后面操作，类名或id分别叫做mask和sidebar。当然首先你的一个ul中的click标签已经准备好,通过它来激发js中的方法,注意我们先给他起个名叫sidebar_trigger，便于后面js中function调用。具体几段代码如下：

/*暗化效果*/ <div class="mask"></div> /*侧拉菜单*/ <div id="sidebar"> <ul> <li><a href="caogao/generic.html">关于侧拉菜单制作</a></li> <li><a href="#">item1</a></li> <li><a href="#">item1</a></li> <li><a href="#">item1</a></li> <li><a href="#">item1</a></li> </ul> </div> /*click链接*/ <div id="menu2"> <ul> <li id="sidebar_trigger"> <a href="#" class="menuToggle">MENU</a> </li> </ul> </div>

Step 3

下面为main.js中的关于mask和sidebar样式处理

.mask { display: none;//隐藏样式 position: fixed; top:0; bottom: 0; right: 0; left: 0; background: rgba(0,0,0,0.3); } #sidebar { position:fixed; top:0; left:-300px;//暂时隐藏于左侧面 bottom: 0; width: 300px; background: #444; color: #fff; padding: 20px 0; transition: left 0.5s;//平滑过渡进入 font-family: "微软雅黑"; } //一些多余样式的清除 #sidebar ul { list-style: none; padding: 0; margin: 0; text-decoration: none; } #sidebar ul a { color:#fff; text-decoration: none; padding: 10px 30px; width: 100%; display: inline-block; } #sidebar ul a:hover { background: #444; text-decoration: none; }

Step 4

接下来就是最至关重要的部分，js方法的实现和中间过程的衔接。不多说，直接上码：

;$(function() { 'use strict'; var sidebar = $('#sidebar'),//选择侧栏 mask=$('.mask'), sidebar_trigger=$('#sidebar_trigger'); /*写两个方法*/ function showSideBar()//点击滑出 { mask.fadeIn(); sidebar.css('left',0);//位置 } function hideSideBar() { mask.fadeOut(); sidebar.css('left',-sidebar.width());//位置 } /*激发*/ sidebar_trigger.on('click',showSideBar) mask.on('click',hideSideBar) })

  • 菜单栏在触发之前该怎么隐藏
  • css样式的一些处理，如transition
